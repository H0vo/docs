<Update label="June 11th, 2025" tags={["Improved"]}>
  **GET /coins**  
  - Added a new `contractAddresses` field in coin data.  
  - Lists smart contract addresses for tokens (e.g., ERC-20).  
  - Empty for native coins like Bitcoin.

  **GET /coins/{coinId}**  
  - Added a new `contractAddresses` field in coin data.  
  - Lists smart contract addresses for tokens (e.g., ERC-20).  
  - Empty for native coins like Bitcoin.
</Update>
